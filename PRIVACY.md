# Privacy Policy / 隐私政策

**VibeCAD** — AI-native conversational CAD (an open-source MCP connector for FreeCAD)

Last updated: 2026-06-10

---

## English

### Summary

VibeCAD runs entirely on your own machine. It does not collect, transmit, or store any personal data.

### Local processing only

All geometry, models, drawings, previews, and exported files (STEP / STL / glTF / PNG) are processed and stored locally on your device. None of your design data is ever uploaded anywhere.

### One-time runtime download

On first use, VibeCAD downloads the FreeCAD runtime (approximately 2–3 GB) from official open-source mirrors (micromamba / conda-forge). This is a plain software download: it does not carry or transmit any personal data.

### No telemetry

VibeCAD collects no telemetry, no usage statistics, and no account information.

### Network access

Network access is limited to software installation: the one-time runtime download described above, and fetching VibeCAD's own open-source dependencies (from PyPI) at install/update time. Your design data is never transmitted.

## 中文

### 概要

VibeCAD 完全在你自己的设备上运行，不收集、不传输、不存储任何个人数据。

### 仅本地处理

所有几何数据、模型、工程图、预览图与导出文件（STEP / STL / glTF / PNG）均只在你的设备上本地处理和存储，任何设计数据都不会被上传。

### 一次性运行时下载

首次使用时，VibeCAD 会从官方开源镜像（micromamba / conda-forge）下载 FreeCAD 运行时（约 2–3 GB）。这是一次普通的软件下载，不携带、不传输任何个人数据。

### 无遥测

VibeCAD 不收集遥测数据、不收集使用统计、不收集账号信息。

### 网络访问

网络访问仅限于软件安装环节：上述一次性运行时下载，以及安装/更新时从 PyPI 获取 VibeCAD 自身的开源依赖。你的设计数据永远不会被传输。
